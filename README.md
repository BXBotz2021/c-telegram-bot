# Telegram Bot in C++

This is a simple Telegram bot written in C++ using the `tgbot-cpp` library. The bot responds to the `/start` command and echoes back any other messages.

## Prerequisites

- C++11 or higher
- CMake
- Boost libraries
- tgbot-cpp library
